🎯 QUICK DEPLOYMENT (3 STEPS):

STEP 1: Run Database Migration
   • Open: https://supabase.com/dashboard
   • SQL Editor → New Query
   • Copy from: run-migration.sql
   • Click: Run
   ✅ Time: 2 minutes

STEP 2: Deploy Supabase Functions
   • Open PowerShell in this folder
   • Run: supabase functions deploy nylas-auth-url
   • Run: supabase functions deploy nylas-callback
   • Set secrets (see RUN_THIS_NOW.md)
   ✅ Time: 5 minutes

STEP 3: Import n8n Workflow
   • Open: https://n8n.instaroute.com
   • Import: workflows/broker-calendar-nylas.json
   • Add Nylas API credential
   • Activate workflow
   ✅ Time: 3 minutes
