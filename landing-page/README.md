# Equity Connect Landing Page

A modern landing page for Equity Connect, connecting homeowners 62+ with trusted reverse mortgage specialists.

## Features

- Dynamic geo-location detection via Vercel Edge middleware
- Responsive design with mobile-first approach
- Modern typography (Space Grotesk + Inter)
- Pre-qualification coordinator profiles
- FAQ section with accordion components

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Vercel Edge Runtime (for geo-location)

## Deployment

Deploy to Vercel with Root Directory set to `landing-page`.
